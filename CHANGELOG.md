# Changelog

All notable changes to detlab. Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/);
this project uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

Substantial expansion past v0.3 — coverage tripled, the Splunk app gained
production-shape features (ES Notables, CIM, RBA), and the portfolio site
grew from 3 pages to 16. Targeted as v0.4.0.

### Added — detections (PRs #4, #5, #6, #7, #9, #10)

- **+11 new cases**, bringing the lab to **24 cases across 14 ATT&CK tactics**:
  T1021.001 RDP lateral, T1021.002 SMB admin shares, T1027.006 HTML smuggling,
  T1068 RPC coercion, T1090.001 internal proxy chaining, T1133 external remote
  services abuse, T1213.002 info-repo bulk read, T1570 lateral tool transfer,
  T1583.001 newly-registered domains, T1595.002 vuln scanning, T1595.003 web
  wordlist scanning.
- Full Enterprise ATT&CK coverage matrix — every tactic now planned or shipped.

### Added — Splunk surface

- **Live-Splunk demo** (`scripts/splunk_demo.py`, `lab/SPLUNK_DEMO.md`):
  HEC ingest + REST saved-search execution for every case. Proves the SPL
  works against a real Splunk, not just the Python mirror in CI. (PR #4)
- **Kill-chain meta-detector** — chains per-case alerts into kill-chain stages. (PR #11)
- **Risk-Based Alerting (RBA)** scoring + analyst triage walkthrough. (PR #12)
- **Splunk ES Notable preview** + **CIM compliance dashboard** +
  **lookup library** page. (PR #15)
- **Open-in-Splunk** deep-link action from any case page. (PR #11)
- **Splunk macro library** with SPL syntax highlighting; dashboard preview;
  alert timeline. (PR #13, #18)

### Added — portfolio site (3 → 16 pages)

Insights area: Kill Chain, Risk, Pyramid of Pain, Threat Groups, CIM,
Macros, Lookups, Schedule heatmap, Data Sources, Roadmap, TacticDetail.
Plus playground detection trace, threshold-slider polish, Insights nav
dropdown, route transitions, About SVG redesign. (PR #5, #6, #9, #11–#18)

### Changed

- Quality audit on the templated cases — fixed SPL `bin _time`, Sigma
  selections, and missing slider knobs from the bulk-shipped batch. (PR #10)
- Backend cleanup — pairwise iteration helpers, dead conditionals dropped,
  exceptions narrowed; shared utility modules extracted from page code. (PR #16, #17)
- `.gitattributes` configured for Linguist so the repo language stats
  reflect detection content, not generated artifacts. (PR #8)

### Verification

- 215 pytest passing (includes cross-case specificity tests).
- ruff clean, TypeScript typecheck clean.

## [0.3.0] — 2026-05-06

### Added

- Portfolio site (`web/`) — Vite + React + TypeScript, Pyodide-powered
  in-browser detection runner, three pages: Coverage / Stats / Case detail.
- **+11 cases** past v0.2: T1572 protocol tunneling, T1090.003 Tor, T1568.002 DGA,
  T1219 RMM, T1190 Suricata exploit, T1046 network service discovery,
  T1110.001 SSH brute force, T1048.003 DNS exfil, T1041 exfil over C2 (chained),
  T1567.002 cloud-storage exfil, T1499.001 volumetric flood.
- Splunk ES integration auto-generated by `build_app.py`:
  `correlationsearches.conf`, `analyticstories.conf`, `eventtypes.conf`,
  `tags.conf`, `workflow_actions.conf`.
- Lab IOC lookups committed: `tor_relays.csv`, `rmm_domains.csv`,
  `cloud_storage_ips.csv`.
- 5 detection styles formalised (aggregation/threshold, timing variance,
  per-record threshold, IOC enrichment, entropy/structure) plus chained
  composition (T1041 consumes `detect_beaconing` output).

### Verification

- 99 pytest passing, ruff clean, TS typecheck clean.
- .spl tarball: 23 KB. Web bundle: 57 KB gzipped initial JS.

## [0.2.0] — 2026-05-04

### Added

- Deployable Splunk app: dashboards (Overview, Detections, ATT&CK Coverage,
  per-case), `nav/default.xml`, savedsearches.
- Build pipeline (`scripts/build_app.py`) — assembles per-case content into
  `.spl` tarball; validates macro references; generates `detlab_cases.csv`
  lookup that powers the dashboards.
- T1071.001 HTTP beaconing (Sliver) case.
