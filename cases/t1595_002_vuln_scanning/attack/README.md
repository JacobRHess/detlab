# Attack reproduction — Vulnerability scanning

Synthetic fixtures live under `tests/`. To regenerate the synthetic
set: `py scripts/generate_fixtures.py`. To capture a real-lab
fixture, deploy the matching tool inside the docker-compose stack
and trim the matching Zeek / Suricata log under `lab/data/`.

## Tools

Nessus, Qualys, Nikto, OpenVAS, sqlmap, nuclei, zgrab

## Safety / scope

- Lab use only.
- The synthetic positive fixture is generated by
  `scripts/generate_fixtures.py` so the test suite is hermetic.
  Replace with a real capture once you have one.
